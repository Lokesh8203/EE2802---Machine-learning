\section*{Class 12, Chapter 6, Exercise 5.19}

Q. Show that of all the rectangles inscribed in a given fixed circle, the square has the maximum area.

\solution The rectangle inscribed in circle is shown below in figure \eqref{fig:Circle}

\begin{figure}[!htb]
    \centering
    \includegraphics[width=\columnwidth]{figs/circle_rectangle.png}
    \caption{Circle}
    \label{fig:Circle}
\end{figure}

\begin{table}[ht!]
    \input{tables/table.tex}
    \caption{points}
    \label{tab:points}
\end{table}

For $0 < \theta < \frac{\pi}{2}$

The area of rectangle inscribed in circle is given by
\begin{align}
    \text{Area} &= \text{AB} \times \text{BC}\\
    & = 2\cos\theta \times 2\sin\theta\\ 
    &= 4\cos\theta\sqrt{1-\cos^2\theta}\\
\end{align}

Let x = $\cos\theta$, area function is given by
\begin{align}
    A\brak{x} &= 4x\sqrt{1-x^2}\\
\end{align}

For $0 < \theta< \frac{\pi}{2}$, i.e. $0 < x < 1$
\begin{align}
    A\brak{x} > 0
\end{align}

So
\begin{align}
    \min_{\lambda} A^2\brak{\lambda} = \min_{\lambda} A\brak{\lambda}
\end{align}

\begin{align}
    A^2\brak{x} &= 16x^2\brak{1-x^2}\\
    &= -16x^4 + 16x^2
\end{align}

This is concave function, so using cvxpy, we get
\begin{align}
    x &= 1.4142 \approx \frac{1}{\sqrt{2}}\\
    x &= \cos\frac{\pi}{4}
\end{align}

So area of square is maximum in all rectangles inscribed in a circle.
